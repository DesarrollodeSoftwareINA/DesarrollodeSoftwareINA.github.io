Why Should You Choose Software Development?

If you want a career with a future, exciting challenges, and a real impact on the world, choose software development. Studying development puts you at the center of technological innovation and opens up the most promising opportunities in today's job market. You will be able to create solutions that improve people's lives, from applications and platforms to revolutionary digital tools. Also, you will have skills that companies highly value, ensuring constant and stable growth. Choose advancement and start building the future today!

Technical Skills You Will Develop in Software Development:

  • Analyze client requirements to develop systems
  • Develop programming logic
  • Design IT solutions and their components using Unified Modeling Language (UML)
  • Manage relational databases using DBMS such as Workbench, phpMyAdmin, PostgreSQL, HeidiSQL, SQL Server Express
  • Develop web applications under client-server architecture using the most in-demand programming languages in the market, such as Python, Java, C#, and frameworks like Angular, React through JavaScript or TypeScript
  • Design and configure a virtual network and its devices using Packet Tracer, applying the most known network topologies
  • Create user manuals using quality standards according to ISO and ITIL standards
  • Implement good software development practices
  • Competence in emerging technologies (artificial intelligence, virtual reality design and manipulation, 3D printing, video mapping)

First Year

In the first year of Software Development, you will learn a variety of topics:

  • Algorithm creation using programming logic
  • Identification of requirements to design IT systems
  • Design of multimedia applications
  • Creation of quality control system manuals implementing ISO standards
  • Web development
  • Conversations in English about IT systems and web development

Second Year

During the second year of Software Development, the following topics will be covered:

  • IT system design
  • Web architecture design
  • Database component programming
  • Object-Oriented Programming (OOP)
  • LAN network design and installation

Third Year

During the third year of Software Development, the following topics will be explored:

  • Development of multimedia software applications to solve various problems
  • Maintenance and assurance of IT system operation
  • Database administration
  • Preparation of IT system documentation
  • Development of components for mobile devices

Our Institution

The National Institute of Apopa (INA) was founded on February 7, 1981, initially operating in the Vicente Acosta School with a single first-year high school section and 44 students. Under the direction of Professor José Jerónimo Yánez and with the support of an administrative team, the institute began its educational work.

In 1984, it moved to its current location, and with the support of the community, additional classrooms were built. By 1988, it offered Academic, Commercial, and Health High Schools, serving 1000 students with 30 teachers.

Since then, the INA has constantly sought new opportunities for students to develop their skills, participate in artistic, sports, and cultural activities, obtaining recognition both nationally and internationally. Since 2015, it has offered a renewed curriculum, including Vocational Administrative Accounting Technical High Schools, Secretariat, Executive Assistance, Community Health, Clinical Laboratory, Technical Nurses, and Software Development.
